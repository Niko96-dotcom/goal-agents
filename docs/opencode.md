# OpenCode `/goal`

Install the plugin from [opencode-goal](https://github.com/Niko96-dotcom/opencode-goal):

```json
"plugin": ["@nikomohr/opencode-goal"]
```

Local development:

```json
"plugin": ["file:///absolute/path/to/opencode-goal/packages/opencode-goal"]
```

Usage: `/goal status`, `/goal <objective>`, `/goal --verify "npm test" …`

See [opencode-goal INSTALL](https://github.com/Niko96-dotcom/opencode-goal/blob/main/docs/INSTALL-OPENCODE.md).
