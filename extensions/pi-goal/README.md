# pi-goal

Pi extension that registers **`/goal`** and bundles the goal skill.

Requires `goal` or `cursor-goal` on PATH (from [opencode-goal](https://github.com/Niko96-dotcom/opencode-goal) `goal-cli` or [cursor-goal](https://github.com/Niko96-dotcom/cursor-goal)).

```bash
pi install /path/to/commandcode-goal/extensions/pi-goal
```

Then in Pi: `/goal status`, `/goal "my objective" --verify "npm test"`.
